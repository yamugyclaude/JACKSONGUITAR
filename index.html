HOW TO USE THIS DATABASE
01
AX8-Edit에서 앰프 선택 후 각 페이지 스크린샷 촬영
Basic → GEQ → Preamp → Power Amp → Power Sply → Speaker → Spkr Drv → Dynamics 순서
02
이 페이지에 스크린샷 이미지 파일로 저장
앰프별 폴더로 정리 — 파일명: 앰프명_페이지명.png
03
Claude에게 이미지 URL 또는 파일 전달
"TX Star Clean Basic 페이지 기본값 데이터화해줘" 방식으로 요청
04
Claude가 파라미터값 읽어서 데이터화 → 아티스트별 변경값 제공
기본값 대비 변경 필요한 파라미터만 정확히 알려주는 방식
