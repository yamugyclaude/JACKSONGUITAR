## 스크린샷 폴더 구조

```

├── index.html                      ← 웹페이지 소스

│       └── dynamics.png
└── Tx Star Clean/                  ← 원본 스크린샷 (임시 보관용)
```

### 앞으로 새 앰프 스크린샷 추가 시
1. `잭슨웹/screenshots/{앰프_폴더명}/` 에 PNG 8장 저장
2. 파일명 규칙: `basic.png`, `geq.png`, `preamp.png`, `power_amp.png`, `power_sply.png`, `speaker.png`, `spkr_drv.png`, `dynamics.png`
3. Claude에게 "앰프명 파라미터 정리해줘" 라고 명령
4. Claude가 이미지 읽기 → 파라미터 추출 → index.html 업데이트 → git push
